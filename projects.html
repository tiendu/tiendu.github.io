depviz

Python · PyPI · Conda · Bioconda · Graph Analysis · CLI · Caching

Inspect dependency graphs for PyPI and Conda/Bioconda environments. depviz resolves package dependencies, calculates dependency blast radius and transitive dependency weight, explains why packages are present, estimates upgrade impact, renders ASCII dependency trees, and caches resolved graphs for fast repeated analysis.

seqgrep

Python · Bioinformatics · FASTA · FASTQ · Gzip · IUPAC · CLI · Multiprocessing

A DNA-aware grep-style search tool for biological sequences. seqgrep supports literal sequence search by default, optional IUPAC nucleotide ambiguity matching, reverse-complement search, circular DNA boundary matches, FASTA/FASTQ input, gzip-compressed files, and chunked multiprocessing for long sequences.

Gommitizen

Go · CLI · Git · Conventional Commits

A Go implementation of Commitizen that helps developers generate conventional commit messages consistently and automate release workflows.

gentr

Go · File Watching · Command Execution · CLI · Developer Tooling

A lightweight command runner that watches files and directories for changes, then automatically reruns a command. watchrun supports recursive watching, glob and stdin-based input, configurable output, and simple install and uninstall commands.
